# Installation

Moongate targets the .NET 10 toolchain pinned in `global.json`, and its server requires an Ultima Online client-data directory at startup. The provided Dockerfile builds a framework-dependent Linux x64 executable on the .NET 10 SDK image and runs it on the .NET 10 runtime image.

Return to the [Server Guide](./index.md), or continue to [Configuration](./configuration.md).
