# Moongate.Templates

`Moongate.Templates` provides the `dotnet new moongate-plugin` template for creating C# plugins
that target the Moongate plugin SDK packages.

Install the template package:

```bash
dotnet new install Moongate.Templates::<version>
```

Create a new plugin:

```bash
dotnet new moongate-plugin --name MyPlugin --plugin-id my-plugin --authors "Squid"
```
